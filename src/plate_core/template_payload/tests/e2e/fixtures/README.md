# Test Fixtures

Fixtures provide reusable setup and teardown for tests. Use this directory for:

- Authentication fixtures (logged-in user)
- Database seeding
- API test data setup
- Custom test contexts

See Playwright documentation on [fixtures](https://playwright.dev/docs/test-fixtures) for details.

## Demo GIFs

GIFs generated from test failure videos using `scripts/gif-from-video.sh` can be placed here.
